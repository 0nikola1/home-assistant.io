Eliqonline

Integrate your ELIQ Online smart meter information into Home Assistant. To get an acess token and the Channel ID, log in to your account.

To enable this sensor in your installation, add the following to your configuration.yaml file:

# Example configuration.yaml entry
sensor:
  platform: eliqonline
  access_token: ACCESS_TOKEN
  name: Power consumption 
  channel_id: CHANNEL_ID

Configuration variables:

  • access_token (Required): The Access Token for your account.
  • channel_id (Optional): Channel ID of your device. Needed if you have more than one device.
  • name (Optional): The name of the sensor, eg. the city.

For details please check the API documentation.
